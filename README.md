# gtile

A simple tiling tool for Linux.

![img](test.png)

## Usage

```
tile [number]
```

This can be mapped directly to the numerical as

```
1   |   2   |   3
    |       |
-------------------
    |       |
7   |   6   |   8

```

and

```
    |       |    
    |       |
4   |   5   |   6
    |       |
    |       |   
```
